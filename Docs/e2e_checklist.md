> 说明：本清单用**最小样例数据**验证每个功能"能跑通"，不追求统计有效性。

## 验证步骤

### 0. 准备
- [ ] 服务器已拉到最新：`cd /opt/ProjectSABC && git pull && docker compose up -d --build`，容器全 Up
- [ ] `admin` 登录 → 首登改密 → 进入仪表盘

### 1. 导入教师（`/admin/import` → 教师名单）
- [ ] 下载模板，按上表填 3 行，删掉示例行 → 上传 → 预览校验通过 → 确认导入
- **预期**：提示新增 3 条；之后 T001/T002/T003 可用 `工号 / 123456` 登录

### 2. 导入学生（`/admin/import` → 学生名单）
- [ ] 同样填 3 行学生 → 导入
- **预期**：新增 3 条；学生可用 `学号 / 123456` 登录

### 3. 课表导入（`admin` → `课表导入`）——课程改为从学校课表导入
- [ ] 选评价学年 `2025-2026`、课程类型 `理论课` → 上传 `25-26-2学期课表.xlsx`
- [ ] 再选 课程类型 `实践课` → 上传 `25-26-3学期课表.xlsx`
- **预期**：每次给出报告——开课 N 门、教师匹配 X 人/新建 Y 人、解析失败行（若有）。课表里"教师"拼音账号若不在教师名单中会自动建（密码 123456）。
- ⚠ 学生名单的"班级"要和课表"上课班级构成"里的班级名一致，否则学生评教匹配不到。

### 4. 教师选参评课程（用课表里某位教师的**拼音账号** / `123456` 登录 → 改密 → `我的参评课程`）
- [ ] 看到该教师本学年的课程列表 → 选一门 →（可改课程类型、勾课程负责人）→ "设为参评课程"
- **预期**：该门标记"当前参评"；学生按其"上课班级"自动关联

### 4-选修. 选修课/无班级课程（教师 → `录入课程/名单`）
课表里选修课常无「上课班级构成」→ 班级匹配不到学生。改用选课名单:
- [ ] 教师在「录入课程/名单」录入该选修课(代码/名称/类型/选修/学年;班级留空)
- [ ] 对该课「管理选课名单」→ 上传名单 Excel(列:学号、姓名)
- **预期**:报告"已加入 X 人 / 已在名单 / 未匹配 Y 人(系统查无的学号会列出)"。名单内学生登录「学生评教」即可看到这门选修课。
- ⚠ 名单只匹配**系统中已存在**的学号,不会新建学生;先确保第 2 步学生已导入。
- 管理员可在「手工课程核查」查看所有手工录入课程及其名单(只读)。

### 4a. 设系主任（`admin` → `系主任设置`）
- [ ] 勾选某教师为系主任（其"所属系部"需填对）
- **预期**：标记"系主任"；该师再次登录左侧出现「质量委员任命」「材料评阅人任命」

### 4b. 系主任任命委员（系主任登录 → `质量委员任命` / `材料评阅人任命`）
- [ ] 从本系教师里勾 1~2 人为质量委员；再勾 1~2 人为材料评阅人
- **预期**：被任命者登录后左侧出现「我的听课对象」/「我的材料评阅对象」

### 4c. 委员自选评价对象（委员登录 → `我的听课对象` / `我的材料评阅对象`）
- [ ] 质量委员：从**全院**老师里勾要听课的（须该师已选参评课程）→ 生成评价任务
- [ ] 材料评阅人：从**本系**老师里勾 → 生成任务
- **预期**：提示"已生成 N 项任务"；未选参评课程的会跳过

### 5. 听课 + 材料评分（委员登录 → `我的评分任务`）
- [ ] 完成"听课""材料审查"5 维度评分并提交
- **预期**：任务变"已完成"

### 6. 现场说课（`admin` → `说课控制台`）
- [ ] 新建场次：加 1 位说课教师 = T001（选 CS101）→ 创建
- [ ] 点"打开大屏"（新标签）；点 T001 行"设为当前"
- [ ] 用 T002、T003 分别访问页面上显示的 `/peer/session/{id}` 地址 → 对 T001 打分提交
- **预期**：每次提交后**大屏实时刷新**（已提交数、去极值均分、5 维度雷达跳动）

### 7. 学生评教（用 **2301001/2301002/2301003** 分别登录改密 → `学生评教`）
- [ ] 每个学生应看到 T001（因班级 数媒2301 匹配 CS101 授课班级）→ 完成匿名问卷
- **预期**：提交后该老师转入"已完成"；不能重复提交

### 8. 个别访谈（`admin` → `访谈管理`）
- [ ] 新建访谈：选 CS101、抽取学生填 `2301001,2301002` → 创建
- [ ] 用 T002、T003 登录 → `我的访谈评分` → 对该访谈打分（能力/方法/考核）
- **预期**：2 名委员打分后访谈状态"已完成"

### 9.（可选）前置限定（`admin` → `前置限定`）
- [ ] 可给某教师录入一条（如"一般教学差错→不得A"）观察对等级的影响；测完可删除

### 10. 全院重算（`admin` → `仪表盘`）
- [ ] 学年填 `2025-2026` → 点"全院重算"
- **预期**：参评教师数、等级分布、排名表出现；T001 有综合分与建议等级（人数少会显示"数据不完整"，正常）

### 11. 会签发布（`admin` → `成绩管理/发布`）
- [ ] 点"发起发布会签"
- [ ] 用 **T002、T003**（审核委员）登录 → `成绩会签` → 都点"同意"
- **预期**：两人都同意后，结果状态变"已发布"，最终等级写入；任一人"驳回"则作废需重发起
- [ ]（可选）测"发起修改等级会签"，同样需两人同意才生效

### 12. 教师看成绩单（用 **T001** 登录 → `我的成绩单`）
- [ ] 发布**前**查 → 显示"成绩评定中"（门禁生效）
- [ ] 发布**后**查 → 综合分 + 三维子项 + **5 维度雷达** + **强弱诊断文字** + 建议/最终等级
- **预期**：未发布看不到、发布后可见且含诊断

### 13.（可选）免计入两级审核（V7.0 校发〔2026〕79号）
- [ ] T001 → `免计入申请`：填学生学号 `2301003` + 理由（≥30 字）→ 提交
- [ ] `admin`/系主任 → `免计入审核`：系部主任→学校质保部 两个标签逐级"同意"
- **预期**：两级全过后该生评价标记免计入；重算后该生那条不计入 T001 学生分

### 14.（可选）申诉
- [ ] T001（成绩已发布）→ `我的申诉`：提交院级申诉
- [ ] `admin` → `申诉处理`（院级）→ 受理或驳回；驳回后 T001 可"申请校级复核"
- **预期**：流程流转正确、各级意见留痕
