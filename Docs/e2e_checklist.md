## 验证步骤

### 0. 准备
- [ ] 服务器已拉到最新：`cd /opt/ProjectSABC && git pull && docker compose up -d --build`，容器全 Up
- [ ] `admin` 登录 → 首登改密 → 进入仪表盘

### 1. 导入教师（`/admin/import` → 教师名单）
- [ ] 下载模板，按上表填 3 行，删掉示例行 → 上传 → 预览校验通过 → 确认导入
- **预期**：提示新增 3 条；之后 T001/T002/T003 可用 `工号 / 123456` 登录

### 2. 导入学生（`/admin/import` → 学生名单）
- [ ] 同样填 3 行学生 → 导入
- **预期**：新增 3 条；学生可用 `学号 / 123456` 登录

### 3. 教师填报课程（用 **T001** 登录 → 改密 → `参评课程填报`）
- [ ] 课程编号 `CS101`、名称 `交互设计基础`、类型 `项目课`、级别 `一般课`、学期 `第二学期`
- [ ] 授课班级：授课专业选 `数字媒体艺术`、授课年级选 `2023级` → 勾选出现的 `数媒2301`（可多选）；勾"我是课程负责人" → 提交
- **预期**：提示填报成功

### 4. 导入监控组名单（`admin` → `监控组名单`）
- [ ] 下载模板，按上表填 T002/T003 两行 → 上传 → 确认导入
- **预期**：提示生成评价任务若干（每人 1 听课 + 1 材料）；无"跳过"则全部成功

### 5. 听课 + 材料评分（用 **T002** 登录改密 → `我的评分任务`；再用 **T003** 重复）
- [ ] T002：完成 CS101 的"听课"和"材料审查"两项评分（5 维度逐题选，提交）
- [ ] T003：同样完成两项
- **预期**：每次提交后该任务变"已完成"；共 2 条听课 + 2 条材料

### 6. 现场说课（`admin` → `说课控制台`）
- [ ] 新建场次：加 1 位说课教师 = T001（选 CS101）→ 创建
- [ ] 点"打开大屏"（新标签）；点 T001 行"设为当前"
- [ ] 用 T002、T003 分别访问页面上显示的 `/peer/session/{id}` 地址 → 对 T001 打分提交
- **预期**：每次提交后**大屏实时刷新**（已提交数、去极值均分、5 维度雷达跳动）

### 7. 学生评教（用 **2301001/2301002/2301003** 分别登录改密 → `学生评教`）
- [ ] 每个学生应看到 T001（因班级 数媒2301 匹配 CS101 授课班级）→ 完成匿名问卷
- **预期**：提交后该老师转入"已完成"；不能重复提交

### 8. 个别访谈（`admin` → `访谈管理`）
- [ ] 新建访谈：选 CS101、抽取学生填 `2301001,2301002` → 创建
- [ ] 用 T002、T003 登录 → `我的访谈评分` → 对该访谈打分（能力/方法/考核）
- **预期**：2 名委员打分后访谈状态"已完成"

### 9.（可选）前置限定（`admin` → `前置限定`）
- [ ] 可给某教师录入一条（如"一般教学差错→不得A"）观察对等级的影响；测完可删除

### 10. 全院重算（`admin` → `仪表盘`）
- [ ] 学年填 `2025-2026` → 点"全院重算"
- **预期**：参评教师数、等级分布、排名表出现；T001 有综合分与建议等级（人数少会显示"数据不完整"，正常）

### 11. 会签发布（`admin` → `成绩管理/发布`）
- [ ] 点"发起发布会签"
- [ ] 用 **T002、T003**（审核委员）登录 → `成绩会签` → 都点"同意"
- **预期**：两人都同意后，结果状态变"已发布"，最终等级写入；任一人"驳回"则作废需重发起
- [ ]（可选）测"发起修改等级会签"，同样需两人同意才生效

### 12. 教师看成绩单（用 **T001** 登录 → `我的成绩单`）
- [ ] 发布**前**查 → 显示"成绩评定中"（门禁生效）
- [ ] 发布**后**查 → 综合分 + 三维子项 + **5 维度雷达** + **强弱诊断文字** + 建议/最终等级
- **预期**：未发布看不到、发布后可见且含诊断

### 13.（可选）免计入三级审核
- [ ] T001 → `免计入申请`：填学生学号 `2301003` + 理由（≥30 字）→ 提交
- [ ] `admin` → `免计入审核`：系部→学院→学校 三个标签逐级"同意"
- **预期**：三级全过后该生评价标记免计入；重算后该生那条不计入 T001 学生分

### 14.（可选）申诉
- [ ] T001（成绩已发布）→ `我的申诉`：提交院级申诉
- [ ] `admin` → `申诉处理`（院级）→ 受理或驳回；驳回后 T001 可"申请校级复核"
- **预期**：流程流转正确、各级意见留痕
